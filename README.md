# Heroes League 🦸‍♂️

🔗 **Démo en ligne** : https://charlylam.github.io/heroes-league/

Projet d’intégration HTML/CSS réalisé dans le cadre de ma formation, axé sur la structure sémantique et la mise en page.

## ✅ Ce que ce projet démontre
- HTML sémantique (sections, navigation, titres)
- Mise en page CSS (Flexbox / Grid selon les sections)
- Organisation des styles par fichiers (base / components)
- Intégration visuelle soignée (typographie, espacements, composants)

## 🧰 Tech stack
- HTML5
- CSS3

## ▶️ Lancer le projet en local
Aucune installation :
1. Cloner le repo
2. Ouvrir `index.html` dans le navigateur

## 🔧 Améliorations prévues
- Responsive design (media queries, ajustement des layouts)
- Accessibilité : états `:focus`, navigation clavier, contrastes
- Optimisation : images et performance
